# YouTube Whisper Q&A
Ask questions about any YouTube video. Transcribe audio, query LLM, get timestamped answers.

Setup:
1. Create venv: python -m venv venv
2. Activate: venv\Scripts\activate
3. Install: pip install -r requirements.txt
4. Add OpenAI key: rename .env.example -> .env
5. Run: python -m app.main
